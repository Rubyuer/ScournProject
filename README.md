# 球探计分

  "球探计分"是一款线上预定球场(篮球/足球)、约球约赛的应用。
  
  1.球场预约
  "球探计分"提供篮球/足球场地的预约。
  
  2.约球约赛
  球场列表可直接寻找组队，若当前球场无组队信息，用户可自行发布组队。
  
  3.世界球场
  罗列出有关于世界的球场，和比赛资讯消息。
  
  4.预定/订单
  约球场订单、约赛订单、我发起的约赛订单、我参与的约赛订单均预定栏目中显示。
  
  5.我的
  显示用户基本信息:头像/昵称;
  我发起的约赛、我参与的约赛、修改登录密码和退出登录等功能。
